listen

  • listen(eventsArray: Array<string | IoEventInfo>): Observable<string | IoEventInfo>[]
    • Defined in subjects/socket-io.ts:105

    A function that receives an array of either strings or IoEventInfo and returns the equivalent subscription after having transformed the input into a @ioEvent and having issued this.listenToEvent.

    Think of this as a bulk listenToEvent that can be deconstructed.

    usage
    
const {helloWorld$, dondeEsLaBiblioteca$} = this.socket.listen([
  'hello-world',
  {name:'where-is-the-library', once: true}
]);

helloWorld$.subscribe(newState => console.debug('helloWorld$',newState));
dondeEsLaBiblioteca$.subscribe(newState => console.debug('dondeEsLaBiblioteca$',newState));

    Parameters

    • eventsArray: Array<string | IoEventInfo>

    Returns Observable<string | IoEventInfo>[]
